# demomim

## estructura

estas son las partes de la web:

- `index.html`: web principal
- `audios/`: web de audios
- `redes/`: web de redes
- ``glosario/`: web de glosario
- `equipo`: web del equipo

los archivos están distribuidos así:

- `css/`: estilos de cada web
- `js/`: scripts de cada web
- `lib/`: bibliotecas de terceros, en este momento YAML y Tone.js
- `datos/`: datos escritos en YAML
- `recursos/`: recursos multimedia, en este momento audios.
